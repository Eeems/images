You can find the source here: https://github.com/Eeems/images

This image contains a base reticulum and lxmf install and by default starts rnsd. You can mount your own configuration folder to `/config`, and it will generate the configuration on first launch.

The docker-compose.yaml included in this folder is an example of the entire stack.
